# MapCompanion

This is a simple iOS app that shows a map with the user’s current location and the corresponding address. The design is inspired by maps found in driving games, with a minimalist and functional approach.

## Description

MapCompanion is a proof of concept developed using AI prompts, since my iOS development knowledge goes back to the Objective-C era and I am not familiar with modern technologies like SwiftUI. All code, including this README and the icons, has been generated entirely by AI.

The app uses SwiftUI to create a modern, native iOS user interface, integrating MapKit to display the map and CLLocationManager to obtain the user’s location.

## Features

- Displays the current location on a map.
- Shows the approximate address.
- Simple, clean design, similar to driving game maps.
- Real-time location updates.

## Technologies Used

- SwiftUI: Framework for the user interface.
- MapKit: For map integration.
- Core Location: For location handling.

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/mikicof/MapCompanion.git
   ```

2. Open the project in Xcode:
   - Open `MapCompanion.xcodeproj` in Xcode.

3. Run the app:
   - Select an iOS simulator or device.
   - Press the play button to build and run.

## Usage

Once installed, the app will open showing the map with your current location. The address will be displayed at the top or bottom of the screen, depending on the layout.

Note: Make sure to grant location permissions when requested.
